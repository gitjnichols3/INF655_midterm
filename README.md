React Task Manager Midterm

This project is a task management application built in React as part of the midterm exam. It builds on concepts from earlier assignments, including components, props, state, lists, keys, and forms.

Features
Add Tasks - Users can enter a task name and description using a form. Both fields are required and validated before submission.
Search Tasks - Tasks can be filtered dynamically using the search input. The search currently applies to task names.
Sort Tasks - Tasks can be sorted alphabetically by name and toggled back to their original order.
Delete Tasks - Each task includes a delete button. A confirmation prompt is displayed before removing the task.
Live Updates - All changes (add, delete, search, sort) update immediately using React state.

Some elements of this application were created in earlier assignments and have been kept in place. The midterm instructions did not specify removing previous functionality, so those components and features remain as part of the project.
